# offLine-test-SF

How to start

$ git clone https://github.com/AaronBHT/offLine-test-SF

$ cd offLine-test-SF

$ npm install

$ npm start

A special case of the requirements:

If users type in '123@test.com' as email, the error information will show 'Email has been used!'.
